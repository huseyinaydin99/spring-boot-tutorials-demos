### client-vuejs

#### Proje klasörünü açma: 
```
cd spring-boot-mongodb-vuejs/src/main/resources/static/client-vuejs/
```

#### Projeyi kurma: 
```
npm install
```

#### Derleme ve sıcak çalıştırma: 
```
npm run serve
```

#### HTTP üzerinde siteye girme: 
```
http://localhost:4200/
```

#### Ürünü derleme: 
```
npm run build
```

#### Dosyaları tamir etme: 
```
npm run lint
```

#### Proje modüllerini silme: 
```
rm -rf node_modules/
```
