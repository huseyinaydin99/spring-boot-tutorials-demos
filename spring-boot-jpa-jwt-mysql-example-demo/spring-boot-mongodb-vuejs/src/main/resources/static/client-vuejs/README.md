# client-vuejs

## Proje klasörünü aç: 
```
cd spring-boot-mongodb-vuejs-example-demo/src/main/resources/static/client-vuejs/
```

## Projeyi kur: 
```
npm install
```

## Derle ve hızlı çalıştır: 
```
npm run serve
```

## HTTP üzerinde projeye bak: 
```
http://localhost:4200/
```

## Ürünü derle: 
```
npm run build
```

## Proje dosyalarını tamir et: 
```
npm run lint
```

## Proje modüllerini sil: 
```
rm -rf node_modules/
```
