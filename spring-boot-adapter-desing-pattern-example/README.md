# spring-boot-adapter-desing-pattern

```
Adapter tasarım deseni, bir nesnenin arayüzünü, başka bir arayüzle uyumlu hale getirerek
birlikte çalışamaz görünen sınıfların birlikte çalışmasını sağlar. Bu desen, mevcut kodun
değiştirilmeden kullanılmasını sağlarken, farklı arayüzleri dönüştürerek uyumluluk sorunlarını çözer.

Genellikle dış servisleri kullanırken, mevcut sistemin arayüzüyle uyumsuz olan 
servislerin entegrasyonunu sağlamak için Adapter tasarım deseni kullanılır.

Function.identity() listedeki her bir elemanın kendisini döndürür.
Yani, listedeki öğeler üzerinde herhangi bir değişiklik yapmadan,
öğelerin doğrudan kendisini haritalama işlemi için kullanır.
```
